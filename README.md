# 🛡️ مِحَكّ الأخلاقيات والنزاهة — Ethics & Integrity Auditor

> 🧠 بطاقة الدماغ (Obsidian): [[_brain/hubs/ethics|ethics]] · [[_brain/00-HOME|HOME]]


مهارة [Claude Skill](https://docs.claude.com/en/docs/agents-and-tools/agent-skills/overview) متخصّصة في **تدقيق جهاز الإفصاح والنزاهة** في البحث العلمي — للرسائل والأطاريح العربية والعراقية أساساً.

مهمّتها الواحدة:

> **كشف الفجوة بين ما يُفصح عنه البحث من أخلاقيات ونزاهة وما هو مُثبَت ومتّسق فعلاً — حضوراً، واتساقاً، وقابليةً للتحقق — ثم إرشاد الإصلاح دون أن تتّهم باحثاً حقيقياً بسوء سلوك.**

هي الحلقة الثامنة في مجموعة تدقيقٍ أكاديميٍّ تحت [bio-colab](https://github.com/bio-colab). لا تُحكّم فوق أخواتها ولا تكرّرهنّ، بل تُضيف طبقةً لا تغطّيها أيٌّ منهنّ: جهاز الإفصاح (الموافقات، تضارب المصالح، التمويل، توافر البيانات، مساهمة المؤلفين، الإفصاح عن الذكاء الاصطناعي، التسجيل المسبق).

## ما الذي يميّزها

### 1. الفرز أولاً — مصفوفة انطباقٍ تمنع إهانة الباحث
لا تُشهر قائمةً غربيةً موحّدةً في وجه كل بحث. تفرز العائلة المعرفية أولاً، ثم تقيس حضورَ كل بندٍ **بمصفوفة عائلته**: لجنةُ الأخلاقيات غيرُ منطبقةٍ على رسالة فقهٍ، وتوافرُ البيانات غيرُ منطبقٍ على نقدٍ أدبيّ. **غيابُ ما لا تتوقّعه العائلةُ ليس عيباً.**

### 2. طبقة حتمية (دون إنترنت، دون تبعيّات)
سكربت `disclosure_scan.py` بـ Python القياسيّ وحده:
- **جرد الإفصاح** — حضورُ البنود الثمانية نصّاً (عربي/إنجليزي)، مصنَّفاً بالمصفوفة.
- **اتساق الإفصاح** — تعارضُ «لا تمويل» مع مموّلٍ مذكور، «متاحة عند الطلب» بلا جهة اتصال، صلاحيةُ بنية DOI/المستودع.
- **بصمات النزاهة** — عبارات مُعذَّبة، قرائن مطابع الورق، تقطيع/نشر مكرّر — **رايات لا أحكام**.

### 3. تحقّقٌ لا تماثليّ وجدارٌ أخلاقيّ أشدّ
تكشف **تعارضاً أو غياباً قابلاً للتحقق**، ولا تُبرّئ ولا تُدين. **سقفُ سلّمها كلِّه «يستوجب تحقّقاً بشرياً/مؤسسياً» — لا «تلفيق» ولا «سرقة» ولا تسمية شخصٍ باتهام.** الخطأ هنا تشهيرٌ لا إزعاج، فالميزان مائلٌ بشدّةٍ نحو التحفّظ.

### 4. تحقّقٌ شبكيٌّ متدرّج + حقنُ وورد جراحيّ
تتحقق من حلّ روابط البيانات والتسجيلات (لا من النيّة)، وتحقن التعليقات في الملف دون تفكيكه.

## الاستعمال السريع

```bash
# فحص ذاتي للسكربت
python scripts/disclosure_scan.py --self-test

# جرد حتميّ (حدّد العائلة للفرز الصحيح)
python scripts/disclosure_scan.py thesis.docx --family quantitative --json out.json
```

ثم اطلب من Claude: «دقّق أخلاقيات ونزاهة هذه الرسالة» أو «افحص بيانات الإفصاح وتوافر البيانات».

## الحدود (تُعلَن دائماً)
لا وصول إلى النيّة ولا إلى ما لم يُكتَب؛ حضورُ البيان لا يثبت صدقه وغيابُه لا يثبت سوء سلوك؛ تغطيةُ قواعد التحقق للعربية ضعيفة؛ الإفصاح قد يكون في موضعٍ لم يُقرأ؛ **الحكم النهائيّ للباحث والمشرف والمؤسسة**. المهارة تُنبّه وتُسنِد، ولا تدين ولا تُبرّئ.
